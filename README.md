# German Beyond Exams

A modern learning platform focused on teaching **German for real-life communication**, not just exam preparation.

This project combines **technology and language learning** to create a structured environment where students can access course recordings, notes, and study material in one place.

---

## Live Demo

Website: *https://germanbeyondexams.de/*

---

## Vision

Many German language courses focus primarily on **passing exams**, but learners often struggle to communicate in **real-life situations**.

**German Beyond Exams** aims to solve this problem by focusing on **practical language learning**, helping students understand how German is used in everyday conversations, workplaces, and real-world environments.

This platform is **designed and developed independently**, combining my **technical knowledge** with my **experience in German language learning**.

---

## Key Features

### Homepage

* Explains the learning philosophy
* Highlights the limitations of exam-focused learning
* Introduces the concept of practical German learning

### Student Dashboard

* Centralized access to learning materials
* Watch **class recordings**
* Access **class notes and study resources**

### Student Learning Panel

* Organized access to course recordings
* Structured study materials for each class

### Admin Dashboard

* Manage courses
* Upload recordings
* Upload learning materials
* Organize student content

## Installation

### 1 Clone the repository

```bash
git clone https://github.com/yourusername/german-beyond-exams.git
```

### 2 Navigate to the project

```bash
cd german-beyond-exams
```

### 3 Install dependencies

Backend

```bash
cd backend
npm install
```

Frontend

```bash
cd frontend
npm install
```

---

### 4 Setup environment variables

Create a `.env` file in the backend folder.

Example:

```
PORT=5000
DATABASE_URL=your_database_connection
```

---

### 5 Run the project

Backend

```bash
npm run dev
```

Frontend

```bash
npm start
```
